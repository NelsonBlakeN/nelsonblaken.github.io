The Letter Game

Graphical team project assigned in the programming course during the 2016 fall semester. The object of the game is to form as many words as possible from a grid of randomly assigned letters of varying size. The game would also save high scores for each difficulty, and display them with the respective players' image and name.

Twitter bot designed to autonomously tweet a .gif image related to the weather forecast for that day. Written using Python and various APIs, and hosted on a Rasperry Pi.
